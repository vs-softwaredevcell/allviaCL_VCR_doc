# VCR Documentation

VCR(Virtual Control Room)은 allviaCL의 핵심 제어 시스템입니다. 이 문서는 VCR의 설치, 구성 및 사용 방법에 대한 포괄적인 가이드를 제공합니다.

## 주요 기능

- 실시간 모니터링 및 제어
- 데이터 수집 및 분석
- 알람 관리
- 리포트 생성

## 시작하기

VCR을 시작하려면 [소개](getting-started/introduction) 문서를 참조하세요.
